## Starter Lumen PHP Framework

## 1. Criando o projeto

git clone https://github.com/mixdinternet/starterlumen.git .

-----------

## 2. Instalando os componentes

$ composer install

-----------

## 3. Configurando o ambiente

$ cp .env.example .env

-----------

## 4. Instalando o gulp

$ npm install

-----------

## 5. Instalado as dependencias de javascript

$ bower install

-----------

## 6. Executando o gulp

$ gulp
